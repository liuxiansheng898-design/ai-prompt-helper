## 📖 使用教程

### 1. 输入主体描述
在输入框中描述你想要生成的内容，例如：
- "一只可爱的橘猫"
- "赛博朋克城市夜景"
- "古装汉服美女"

### 2. 选择类型和风格
- **类型**：文生图 / 文生视频
- **风格**：写实、动漫、电影、中国风、奇幻、科幻

### 3. （可选）高级选项
- 质量修饰：8K超高清、大师级作品
- 光线设置：黄金时刻、电影布光

### 4. 生成并复制
点击"一键生成提示词"，复制生成的提示词到AI工具中使用。
